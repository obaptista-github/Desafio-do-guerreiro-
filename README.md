## Desafio do guerreiro 

# Para resolver esse exercício:

- vamos criar uma classe Heroi que terá as propriedades nome,
- idade e tipo,

- que exibe uma mensagem diferente dependendo do tipo do herói.

# Seguindo as instruções:
